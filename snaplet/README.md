# snaplet

A fresh repository scaffold for `snaplet`.

## Quick start

1. Create a new GitHub repository named `snaplet`.
2. Copy this folder into the new repo root.
3. Initialize git and push:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin git@github.com:<your-user>/snaplet.git
git push -u origin main
```

## Included files

- `.gitignore` for Node.js/TypeScript projects.
- `package.json` with starter scripts.
